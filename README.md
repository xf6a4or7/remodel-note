# remodel-note

> 인테리어 상담 녹취 → STT → 교정 → 분석 → 회의록/요구사항 자동화

인테리어·리모델링 현장의 고객 미팅·통화 녹취를 자동으로 문서화하고 분석하는 파이프라인입니다.

> ⚙️ 구조와 코드는 곧 업로드 예정입니다.

## 파이프라인 목표 흐름

```
음성 파일 (m4a 등)
  → STT 변환            (네이버 CLOVA Speech API, 화자분리 포함)
  → 현장 용어 오인식 교정 (용어 사전 + Claude 문맥 교정)
  → 구조화 분석          (공간별·자재별 고객 요구사항 태깅)
  → 산출물 생성
        ├─ 회의록   (.docx)
        ├─ 요구사항 (.json)
        └─ 요약     (.md)
```
